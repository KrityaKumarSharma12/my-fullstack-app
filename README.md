# Full Stack App 🚀

My first full stack web application using modern technologies.

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| Frontend | React.js, Tailwind CSS, Vite |
| Backend | Node.js, Express.js |
| Database | PostgreSQL, Prisma ORM |

## ✨ Features

- ✅ Create and view users
- ✅ Create posts linked to users
- ✅ RESTful API design
- ✅ Responsive modern UI

## 🚀 How to Run Locally

### Prerequisites
- Node.js installed
- PostgreSQL installed

### Backend Setup
```bash
cd backend
npm install
npx prisma migrate dev
npm run dev
